# 无名杀扩展项目

无名杀（NoName）游戏引擎扩展合集。

## 安装方法

将本项目复制到无名杀的扩展目录，游戏启动时自动加载。

## 目录结构

```
搬山道士/
├── audio/          # 音频文件（技能音效、背景音乐）
├── card/           # 自定义卡牌
│   └── dgch/       # 大鬼畜扩展
├── character/      # 角色扩展
│   ├── huluwa/     # 葫芦娃扩展
│   ├── net/        # 网络梗角色扩展
│   ├── other/      # 其他角色扩展
│   ├── tv/         # 电视剧角色扩展
│   ├── wezy/       # 万恶之源扩展（原创角色）
│   └── yxsj/       # 异形世界扩展
├── css/            # 样式文件
├── image/          # 图片资源（角色头像、卡牌图片）
├── main/           # 主配置文件
│   ├── config.js       # 扩展配置
│   ├── content.js      # 内容加载
│   ├── precontent.js   # 预加载配置
│   └── utils.js        # 工具函数
├── extension.js    # 扩展入口文件
├── info.json       # 扩展信息
└── README.md       # 项目说明
```

## 技术特点

- ES6+ 语法，async/await 异步处理
- 遵循无名杀官方技能编写规范(也可能不遵循)
- 一点也不完整的 AI 配置
- 多语言支持(ai生成的, 别听他扯淡)

## 开发说明


### 文件结构

每个角色扩展包含以下文件：

- `character.js`：角色定义
- `skill.js`：技能定义
- `translate.js`：翻译文件
- `intro.js`：角色介绍
- `index.js`：扩展入口
